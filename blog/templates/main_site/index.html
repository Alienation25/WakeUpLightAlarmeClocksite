Wake-up light alarm clock

‹ ›

Pros of using

1) time in 12 hour format
2) signal time
3) temperature

Clock .

Simple and user-friendly design. Possibility of setting the time and setting the backlight.A cable is used for charging micro USB

Temperature

Also, the watch can determine the temperature in the room and have the ability to control the brightness of the LED strip.To measure the temperature, we used a tmp 36 sensor, the alternatives of which are dcht 11, lm 35, bme barometer Illumination.To measure the ambient light, we used a photoresistor instead of which a photodiode and a phototransistor could be used. This was done to implement the blooming function.

Alarm clock

To control the time, we used tact buttons, instead of which it was possible to use a potentiometer or touch buttons.
